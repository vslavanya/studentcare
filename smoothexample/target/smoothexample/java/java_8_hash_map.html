HashMapIteration.java

Following example shows how to iterate through a HashMap using forEach and lambda expressions. 
package com.smoothexample.java8;

import java.util.HashMap;
import java.util.Map;


public class HashMapIteration 
{
    public static void main( String[] args )
    {
        Map map = populateMap();
        
        map.forEach((key,value)->{
        	System.out.print("key ="+key);
        	System.out.println("  value ="+value);
        });
    }

	private static Map populateMap() {
		 Map map = new HashMap<>();
		 map.put("name", "John");
		 map.put("country", "US");
		 map.put("state", "NJ");
		 
		 return map;
	}
}

Sample output

Following output will be printed when running the above example. 
key =country  value =US
key =name  value =John
key =state  value =NJ
